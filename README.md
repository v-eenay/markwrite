## 📋 Overview

MarkWrite is a professional-grade Markdown editor designed for writers, developers, and content creators who value simplicity and productivity. With its real-time preview and intuitive interface, you can focus on creating beautiful documents without distractions.

The application combines modern web technologies with a thoughtful user experience to provide a seamless writing environment for creating well-formatted documents that can be exported to industry-standard formats.

## ✨ Features

### User Interface
- **Professional, Clean Interface** — Distraction-free environment optimized for content creation
- **Split-Pane View** — Edit Markdown on one side and see the rendered preview in real-time
- **Responsive Design** — Fully functional on both desktop and mobile devices
- **Dark/Light Mode** — Toggle between themes for comfortable editing in any environment
- **High-Contrast UI** — Accessible design with carefully selected color schemes

### Editor Capabilities
- **Markdown Syntax Highlighting** — Visual cues for Markdown syntax elements
- **Code Syntax Highlighting** — Support for multiple programming languages in code blocks
- **Formatting Toolbar** — Quick access to common Markdown formatting options
- **Synchronized Scrolling** — Editor and preview panes scroll together for seamless editing

### Document Management
- **PDF Export** — Generate professional PDFs with proper formatting and high-quality text rendering
- **DOCX Export** — Create Microsoft Word documents with preserved formatting
- **Page Breaks** — Control document flow with custom page breaks for printed output
- **Consistent Export Formatting** — Dark text on white background in exports regardless of theme

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or newer)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/v-eenay/markwrite.git
cd markwrite

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Building for Production

```bash
# Create optimized production build
npm run build

# Preview the production build locally
npm run preview
```

## 📖 Usage Guide

### Basic Workflow
1. Write Markdown in the left editor pane
2. See the formatted output in the right preview pane
3. Use the toolbar buttons for quick formatting options
4. Adjust the split view by dragging the divider
5. Toggle between light and dark mode using the theme button
6. Export your document when finished

### Markdown Features
- Headers (H1-H6)
- Text formatting (bold, italic, strikethrough)
- Lists (ordered and unordered)
- Code blocks with syntax highlighting
- Links and images
- Blockquotes
- Tables
- Page breaks for exported documents

### Export Options
- **PDF Export** — Professional PDFs with proper formatting
- **DOCX Export** — Microsoft Word documents with preserved formatting
- **Page Breaks** — Add `---pagebreak---` on its own line to create page breaks

## 🛠️ Technology Stack

### Frontend
- **React 19** — Modern UI library with hooks and functional components
- **Vite 6** — Fast build tool and development server
- **Tailwind CSS** — Utility-first CSS framework for responsive design

### Editor Components
- **CodeMirror 6** — Advanced text editor with syntax highlighting
- **Marked** — Markdown parsing and rendering
- **Highlight.js** — Syntax highlighting for code blocks
- **React-Split** — Split-pane functionality

### Document Processing
- **html2pdf.js** — PDF generation with customizable options
- **docx** — Word document generation with formatting support
- **html2canvas** — HTML to canvas conversion for exports
